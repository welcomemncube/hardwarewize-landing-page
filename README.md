# Hardwize Landing Page

Compare hardware prices across South Africa. Find the best deals on cement, tools, paint, and building materials from Builders, Build It, Cashbuild, Leroy Merlin & more — all in one app.

## Overview

This is the official landing page for **Hardwize**, a mobile app that lets you compare hardware prices across South Africa's major retailers. Built with pure HTML, CSS, and JavaScript — no frameworks needed.

## Features

- Responsive landing page with modern design
- Interactive phone mockup showcasing the app UI
- Partner store listings
- Feature cards, testimonials, and call-to-action sections
- Smooth scroll navigation with sticky navbar

## Getting Started

Simply open `index.html` in any modern web browser:

```bash
open index.html
```

Or serve it locally with any static file server:

```bash
npx serve .
```

This is a static site. Deploy to any hosting platform (GitHub Pages, Netlify, Vercel, Cloudflare Pages, etc.) by pointing it to the `index.html` file.
